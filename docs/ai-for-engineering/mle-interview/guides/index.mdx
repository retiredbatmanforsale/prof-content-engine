These guides translate vague hiring loops into concrete preparation. Each guide answers four questions for a specific (company, level):

1. **Rounds** — what stages exist, how long, what format
2. **Evaluation criteria** — what each interviewer is scoring
3. **Common questions** — recently asked prompts grouped by type
4. **Preparation focus** — where to spend time given the rubric weighting

:::tip[How to use these guides]
Pick the (company, level) closest to your target. Read top to bottom once, then return to the round most likely to gate you.
:::
