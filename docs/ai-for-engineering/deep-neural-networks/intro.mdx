# Deep Neural Networks

This course builds **deep neural networks** from the ground up: the **smallest mathematical units** (**neurons**), how they relate to **linear** and **logistic regression**, then **layers**, **depth**, **training** with gradients, practical patterns for **dense** architectures, and a **NumPy** **from-scratch** capstone tied to the **[lexai-fellowship](https://github.com/retiredbatmanforsale/lexai-fellowship)** notebook.

:::note[Prerequisites]

- **Foundations of Regression** — especially gradient descent, the sigmoid, and SSE vs MLE
- Comfort with vector and matrix notation (dot products, transpose, matrix multiplication)
- Python basics for the NumPy capstone (lesson 10) — earlier lessons are equation-driven and don't require code

:::
