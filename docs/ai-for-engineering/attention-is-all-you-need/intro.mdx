# Attention Is All You Need — the 2017 paper

This course is a close reading of [Vaswani et al. (2017), *Attention Is All You Need*](https://arxiv.org/abs/1706.03762) — the paper that introduced the **encoder–decoder transformer** for machine translation. We focus on what the paper actually proposed: **self-attention**, **multi-head attention**, **cross-attention**, and the two-tower architecture in which an encoder reads the source sequence and a decoder generates the target while attending to encoder memory.

For the modern decoder-only world (GPT-2, GPT-3, the chat models you use every day), see the companion course **Build and Train Your Own GPT-2**, which builds a decoder-only transformer end-to-end in PyTorch.

:::note[Prerequisites]

- **Deep Sequence Modelling (RNN)** — you'll get the most from this course if you've seen the RNN family first; lesson 1 sets up *why* this paper happened.
- **Deep Neural Networks** — forward pass, backprop, softmax, matrix multiplication.
- Linear algebra: dot products, vector norms.

:::
