# Foundations of Regression

This course covers **linear regression**, **logistic regression**, the **sigmoid**, **decision boundaries**, and **MLE** (why not SSE for probabilities).

:::note[Prerequisites]

- Comfort with high-school algebra (lines, slopes, exponents)
- Basic Python helps but is not required — equations and intuition come first
- No machine learning background assumed — this is the recommended starting course on the Engineering track

:::
