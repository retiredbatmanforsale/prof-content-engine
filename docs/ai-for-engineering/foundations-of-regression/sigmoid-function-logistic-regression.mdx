## Mathematical definition

$$
\sigma(z) = \frac{1}{1 + e^{-z}}
$$

## Key properties of the sigmoid function

### Bounds the output between 0 and 1

$$
\text{As } z \to -\infty,\quad \sigma(z) \to 0
\qquad\text{(very large negative } z\text{)}
$$

$$
\text{As } z \to +\infty,\quad \sigma(z) \to 1
\qquad\text{(very large positive } z\text{)}
$$

$$
\text{When } z = 0,\quad \sigma(z) = \frac{1}{2} = 0.5
$$

### Smooth and differentiable

This makes it <strong>perfect for optimization</strong> using gradient descent.

### Non-linear transformation

It <strong>allows us to model non-linear relationships</strong> in the data in the sense that the <em>probability</em> in **x** is a non-linear function even when **z** is linear in **x**.

### Centered at 0.5

When <strong>z = 0</strong>, <strong>σ(z) = 0.5</strong> — the decision threshold between &quot;more class 0&quot; vs &quot;more class 1&quot; in the usual binary setup.

## Intuitive understanding

Think of the sigmoid function as a <strong>&quot;squashing&quot;</strong> mechanism:

<div className="not-prose my-6 grid gap-3 sm:grid-cols-3">
  <div className="rounded-xl border-2 border-emerald-300 bg-emerald-50/90 p-4 dark:border-emerald-800 dark:bg-emerald-950/30">
    <p className="m-0 text-sm font-bold text-emerald-900 dark:text-emerald-200">Large positive z</p>
    <p className="mt-2 mb-0 text-sm text-slate-800 dark:text-slate-200">Gets <strong>squashed toward 1</strong>.</p>
  </div>
  <div className="rounded-xl border-2 border-slate-300 bg-slate-50/90 p-4 dark:border-slate-600 dark:bg-slate-900/40">
    <p className="m-0 text-sm font-bold text-slate-800 dark:text-slate-200">z near 0</p>
    <p className="mt-2 mb-0 text-sm text-slate-800 dark:text-slate-200">Outputs land <strong>near 0.5</strong>.</p>
  </div>
  <div className="rounded-xl border-2 border-rose-300 bg-rose-50/90 p-4 dark:border-rose-800 dark:bg-rose-950/30">
    <p className="m-0 text-sm font-bold text-rose-900 dark:text-rose-200">Large negative z</p>
    <p className="mt-2 mb-0 text-sm text-slate-800 dark:text-slate-200">Gets <strong>squashed toward 0</strong>.</p>
  </div>
</div>

This means that even if our linear model produces a value like <strong>27</strong> or <strong>−15</strong>, the sigmoid function will convert it to a <strong>probability between 0 and 1</strong>.
