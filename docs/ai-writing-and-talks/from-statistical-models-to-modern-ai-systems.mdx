## AlexNet — why GPUs mattered

| Ingredient | Why it mattered |
| ---------- | ---------------- |
| **~8-layer CNN** | Strong signal on **ImageNet** (hard, real-world vision) |
| **Two GPUs** | Split model across devices → **parallel** matmuls |
| **Graphics mindset** | Pixels / regions → **independent** parallel work |

## Moore’s Law vs “accelerated computing”

| Moore’s Law (classic intuition) | AI era add-on |
| ------------------------------ | ------------- |
| ~**2×** capability / **~2 years** | **GPUs / TPUs** = throughput for **dense linear algebra** |
| Cost per transistor falls | **“Moore’s law on steroids”** for **parallel** training |

**Order-of-magnitude story (talk track, not a benchmark):**

- Early GPU-class models on the order of **~10⁸** parameters  
- Frontier systems today in the **trillions** of parameters (with caveats on counting MoEs, etc.)

## Modern AI — where the problems moved

| Theme | What changed |
| ----- | ------------- |
| **Architecture** | Transformers dominate language &amp; increasingly vision/multimodal |
| **Performance** | Models exceed humans on **narrow** suites—forces honest evaluation |
| **Next frontier** | **Reasoning**, causality, **physics** — models that reflect how the world works |
| **World models** | Learn from **observation** of environment, not text alone |

**Speaker note:** “We are in a **fix and extend** phase—making systems **reliable**, **grounded**, and **interpretable** enough for real stakes.”
