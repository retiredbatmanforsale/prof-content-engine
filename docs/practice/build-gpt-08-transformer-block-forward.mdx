# Helpers from earlier problems (provided so you can focus on the block wiring):
def split_heads(x, n_head):
    B, T, C = x.shape; D = C // n_head
    return x.reshape(B, T, n_head, D).transpose(0, 2, 1, 3)

def combine_heads(x):
    B, H, T, D = x.shape
    return x.transpose(0, 2, 1, 3).reshape(B, T, H * D)

def attention_batched(Q, K, V, mask=None):
    d = Q.shape[-1]
    scores = Q @ K.swapaxes(-2, -1) / np.sqrt(d)
    if mask is not None:
        scores = np.where(mask, scores, -1e9)
    s = scores - scores.max(axis=-1, keepdims=True)
    e = np.exp(s); w = e / e.sum(axis=-1, keepdims=True)
    return w @ V

def layer_norm(x, gamma, beta, eps=1e-5):
    mu = x.mean(axis=-1, keepdims=True)
    var = x.var(axis=-1, keepdims=True)
    return gamma * (x - mu) / np.sqrt(var + eps) + beta

def multi_head_attention(x, W_qkv, W_o, n_head, mask=None):
    qkv = x @ W_qkv
    Q, K, V = np.split(qkv, 3, axis=-1)
    Q = split_heads(Q, n_head); K = split_heads(K, n_head); V = split_heads(V, n_head)
    out = attention_batched(Q, K, V, mask=mask)
    return combine_heads(out) @ W_o

def gelu(x):
    return 0.5 * x * (1.0 + np.tanh(np.sqrt(2.0 / np.pi) * (x + 0.044715 * x ** 3)))

def mlp(x, W1, W2):
    return gelu(x @ W1) @ W2

import numpy as np

def transformer_block(x, params, n_head, mask=None):
    p = params
    # Sub-layer 1: pre-LN -> attention -> residual
    h = x + multi_head_attention(
        layer_norm(x, p['gamma1'], p['beta1']),
        p['W_qkv'], p['W_o'], n_head, mask=mask,
    )
    # Sub-layer 2: pre-LN -> MLP -> residual
    out = h + mlp(
        layer_norm(h, p['gamma2'], p['beta2']),
        p['W_mlp1'], p['W_mlp2'],
    )
    return out
`}

The GPT series capstone. Wire together every component you've built — LayerNorm (problem 07), multi-head attention (06, which itself depends on 03/04/05) — plus an MLP into the canonical pre-LN transformer block. This is the unit GPT-2 stacks twelve times.

The structure (two sub-layers, two residuals):

`h    = x + multi_head_attention(layer_norm(x, gamma1, beta1))`

`out  = h + mlp(layer_norm(h, gamma2, beta2))`

Why **pre-LN** (LN inside the residual) and not post-LN (LN around the residual)? Pre-LN keeps the residual path an exact identity — gradients flow back through the skip connection unchanged. Post-LN puts LayerNorm in the gradient path, which destabilises very deep stacks. GPT-2, GPT-3, and Llama all use pre-LN.

The diagnostic the harness runs: zero out the four projection matrices and verify that the block becomes the identity — `block(x) == x` to numerical precision. This only passes if **both** residual sums are wired: zeroed projections make `attn(...)` and `mlp(...)` both produce zeros, so `x + 0 + 0 == x`.

After this problem, you've implemented every piece of a single GPT-2 layer. Stack twelve, add token + positional embeddings on the input and an unembedding+softmax on the output — and you have GPT-2 small.
