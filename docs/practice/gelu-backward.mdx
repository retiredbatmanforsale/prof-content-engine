import numpy as np

def gelu_backward(grad_out, x):
    c = np.sqrt(2.0 / np.pi)
    g = c * (x + 0.044715 * x ** 3)
    tanh_g = np.tanh(g)
    sech2 = 1.0 - tanh_g ** 2                    # sech^2(g) = 1 - tanh^2(g)
    g_prime = c * (1.0 + 3.0 * 0.044715 * x ** 2)
    dgelu = 0.5 * (1.0 + tanh_g) + 0.5 * x * sech2 * g_prime
    return grad_out * dgelu
`}

>

Derive and implement the gradient of GELU (tanh approximation). The forward — `0.5 · x · (1 + tanh(g(x)))` — is one line; the backward is the product rule plus the chain rule on the tanh.

The math:

`g(x)  = sqrt(2/π) · (x + 0.044715 · x³)`

`g'(x) = sqrt(2/π) · (1 + 3 · 0.044715 · x²)`

`d_GELU/dx = 0.5 · (1 + tanh(g)) + 0.5 · x · (1 − tanh²(g)) · g'(x)`

Multiply by the upstream `grad_out` to get the result.

Two correctness checks the harness probes:

1. **Finite-difference cross-check.** The analytic gradient must match the numerical gradient computed from the forward pass within `1e-3`. This catches every sign error and missing chain-rule term. (And there are several places to slip up — five separate `x`s appear in the derivation.)
2. **Limits.** At `x → +∞`, `tanh(g) → 1` and the second term decays, leaving `d/dx → 1`. At `x → −∞`, `tanh(g) → -1`, both terms collapse to 0. Useful smoke checks.

Reuse `tanh(g(x))` — calling `np.tanh` is the expensive part and you need it for both `(1 + tanh)` and `(1 − tanh²)`.
