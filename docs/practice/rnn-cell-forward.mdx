The vanilla RNN cell — one timestep of the recurrent computation. By itself, simple. Composed across hundreds of timesteps, it produces the vanishing-gradient pathology that LSTMs and GRUs were built to fix, and that transformers sidestepped entirely with attention.

The math:

`h_t = tanh(x_t · W_x + h_{t-1} · W_h + b)`

Three things the harness checks:

1. **Shape and broadcasting.** Bias `(H,)` must broadcast across the `(B, H)` batch.
2. **Tanh saturation.** Values are bounded in `(-1, 1)` regardless of input magnitude. This is what keeps the hidden state from blowing up across timesteps — but also what makes gradient ≈ `1 - tanh²(z)` saturate to ~0 for large `|z|`, killing learning at depth.
3. **Identity recurrence.** With `W_x=0, W_h=I, b=0` and small `h_prev`, the cell approximates the identity (`tanh(small)` ≈ `small`). A useful sanity check that you didn't drop a term.

Production sequence models stopped using vanilla RNNs around 2014 (LSTMs took over) and stopped using LSTMs around 2018 (transformers took over). But the cell remains a core ML interview question — and a reference point for understanding what attention solved.
