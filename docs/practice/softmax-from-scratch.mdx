export const SOLUTION = `import numpy as np

def softmax(x):
    # Subtract the max to keep the largest exponent at 0 — same result, no overflow.
    shifted = x - np.max(x)
    exps = np.exp(shifted)
    return exps / exps.sum()
`;

The softmax function turns a vector of arbitrary real numbers into a probability distribution. It's the final layer of every classifier, the gate at the top of every transformer head, and the operation that turns logits into next-token probabilities.

Implement it from scratch in numpy. Your function should:

- Take a 1-D array of real numbers.
- Return an array of the same shape where every element is in `(0, 1)` and the array sums to 1.
- **Be numerically stable.** A naïve implementation (`exp(x) / exp(x).sum()`) overflows for inputs in the thousands, which is exactly the regime real LLM logits live in. The fix is one line — but you have to know it.

The mathematical definition:

`softmax(x)_i = exp(x_i) / sum_j exp(x_j)`

The numerical-stability trick: `softmax(x) == softmax(x - c)` for any constant `c`, because the constant cancels in the ratio. Picking `c = max(x)` guarantees the largest exponent is `e^0 = 1`, no overflow.
