import numpy as np

def attention(Q, K, V, mask=None):
    d_k = Q.shape[-1]
    scores = Q @ K.T / np.sqrt(d_k)
    if mask is not None:
        scores = np.where(mask, scores, -1e9)
    # numerically-stable softmax over the last axis
    scores = scores - scores.max(axis=-1, keepdims=True)
    weights = np.exp(scores)
    weights = weights / weights.sum(axis=-1, keepdims=True)
    return weights @ V
`}

Scaled dot-product attention is the single most important operation in modern deep learning. Every transformer head — self, cross, causal, multi — bottoms out in this function.

The formula:

$$
\\text{Attention}(Q, K, V) = \\text{softmax}\\!\\left(\\frac{QK^\\top}{\\sqrt{d_k}}\\right) V
$$

Three subtleties that interview problems probe for:

1. **The $\\sqrt{d_k}$ scaling matters.** Without it, the dot products grow with the dimension and softmax pushes all the mass onto a single key, killing gradients.
2. **Masks are applied to the *scores*, not the weights.** Set blocked positions to a large negative number *before* softmax, so they become ~0 *after* softmax. Setting them to 0 after the fact would re-normalize the wrong way.
3. **Numerical stability.** The same max-shift trick from softmax applies here — subtract the row-max from the scores before exp.

Implement it. The harness compares your output against a reference numpy implementation across several shapes, with and without a causal mask.
