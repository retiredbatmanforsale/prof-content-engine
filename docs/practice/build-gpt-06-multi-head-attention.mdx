def attention_batched(Q, K, V, mask=None):
    # Vectorised scaled dot-product attention. Works on any leading dims.
    # Q, K, V have shape (..., T, D). Returns (..., T, D).
    d = Q.shape[-1]
    scores = Q @ K.swapaxes(-2, -1) / np.sqrt(d)
    if mask is not None:

import numpy as np

def multi_head_attention(x, W_qkv, W_o, n_head, mask=None):
    # 1) Project to Q, K, V (fused).
    qkv = x @ W_qkv                               # (B, T, 3C)
    Q, K, V = np.split(qkv, 3, axis=-1)           # each (B, T, C)

    # 2) Split each into heads.
    Q = split_heads(Q, n_head)                    # (B, H, T, D)
    K = split_heads(K, n_head)
    V = split_heads(V, n_head)

    # 3) Scaled dot-product attention per head.
    out = attention_batched(Q, K, V, mask=mask)   # (B, H, T, D)

    # 4) Recombine heads and project out.
    out = combine_heads(out)                      # (B, T, C)
    return out @ W_o                              # (B, T, C)
`}

The full Multi-Head Attention layer — the unit that gets stacked twelve times in GPT-2 small. This problem composes everything from earlier in the series: the projection, head split (problem 05), single-head attention (problem 03), the optional causal mask (problem 04), head combine (problem 05), output projection.

The shapes and parameter budget:

`x:      (B, T, C)`

`W_qkv:  (C, 3C)`  → 3·C² params (fused Q, K, V projections)

`W_o:    (C, C)`   → C² params (output projection)

**Total parameters per attention block: 4 · C².** For GPT-2 small (C = 768) that's 2.36M per block × 12 blocks = ~28M, about 23% of the 124M-parameter model.

Helpers are provided in the starter — your job is the five-line wiring that turns a `(B, T, C)` input plus the projection matrices into a `(B, T, C)` output. The mask (if given) flows through unchanged; the causal triangular mask is the standard choice for autoregressive language modelling.
