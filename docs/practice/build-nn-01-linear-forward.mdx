import numpy as np

def linear_forward(x, W, b):
    return x @ W + b
`}

`linear_forward` — sometimes called `dense`, `fully_connected`, or `nn.Linear` in PyTorch — is the bedrock of every feed-forward network. You'll wire it together with non-linearities and a backward pass through the rest of this series until you're training a 2-layer MLP on XOR by problem 7.

Implement it. The math is one line:

`y = x @ W + b`

Three things the harness checks:

1. **Shape:** `(B, in)` → `(B, out)`.
2. **Correctness:** matches `x @ W + b` to numerical precision.
3. **Bias broadcasting:** the bias `(out,)` adds correctly across the batch dimension — no reshape gymnastics needed thanks to NumPy's broadcasting.

This is the simplest problem in the series. The work in `build-nn-05-linear-backward` is harder — but only after you've nailed the forward pass.
